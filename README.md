# Arch-Config
My very simple arch config for hyprland using noctalia.

Includes my basic hyprland.lua as well as my other configs I use nothing crazy at all just super basic revolving around having noctalia shell.

Using the latest version of Hyprland with the .lua config as well as noctalia v5 their unstable branch nothing special for noctalia just installed it and set it up in the hyprland.lua no configuration just small visual tweaks made to my liking.

This repo was just an easy way to store my configuration files to get them anywhere I need.

There is no noctalia config so it'll be a blank slate for you to make your own.

Just download and unzip the files copy them to your .config folder except for the Icons folder rename it to .icons and place it in your home directory I use nautilus for files the theme I use for that can be found here https://www.gnome-look.org/p/2346341 unzip it and place it in the .icons folder as well I use nwg-look to set the themes for gtk.

I also inclueded the wallpaper I like to use just copy it to your wallpapers folder.

I also use hyprlands scrolling mode as I like it over the regualr dwindle mode. I have all the modes listed in the configuration so it can easily be switched to any of them.
